# tanaid

## Setup

Install Rust with `rustup`:

```sh
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Run any Cargo command from the repo root to install/use the toolchain:

```sh
cargo check
```

## Development

Run the compiler checks with:

```sh
cargo check --all-targets
```
